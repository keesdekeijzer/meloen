# Voorbeeld van tag bestand
Compleet voorbeeld, er hoeft verder niets in het bestand te staan

```
---
layout: tagpage
title: "Tag: eerste"
tag: eerste
---
```
# voorbeeld van post
Dit is alleen de kop, pas ook de datum/tijd aan

```
---
layout: post
title: Eerste post
date: 2021-12-05T00:00:00+00:00
tags: eerste post
description: de allereerste post
---
```
